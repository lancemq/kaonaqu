# 2026年本市义务教育阶段学校招生入学实施意见

> **摘要**：上海市教委发布2026年义务教育招生入学实施意见，政策与2025年保持一致，4月10日开通入学报名系统。
>
> **来源**：上海市教育委员会  
> **发布日期**：2026-03-26  
> **适用对象**：幼升小、小升初学生及家长

---

2026年3月26日，上海市教育委员会发布《2026年本市义务教育阶段学校招生入学工作实施意见》，明确今年招生政策总体保持稳定。

## 一、总体原则

### 免试就近入学
- 坚持"免试就近入学"原则
- 严禁以各类考试、竞赛、培训成绩或证书证明等作为招生依据
- 严禁以面试、评测等名义选拔学生

### 公民同招
- 公办学校和民办学校同步招生
- 民办学校报名人数超过招生计划的，实行电脑随机录取（摇号）
- 未被民办录取的，由区教育行政部门统筹安排公办入学

## 二、关键时间节点

| 时间 | 事项 |
|------|------|
| 4月7日 | 各区公布义务教育招生入学实施细则 |
| 4月10日 | 开通"一网通办"和"上海市义务教育入学报名系统" |
| 4月10日-23日 | 小学入学信息登记 |
| 4月10日-23日 | 初中入学信息核对 |
| 4月26日-28日 | 民办小学网上报名 |
| 4月26日-28日 | 民办初中网上报名 |
| 5月12日-13日 | 民办小学、初中电脑随机录取 |
| 5月17日-18日 | 民办学校录取验证 |
| 5月20日起 | 公办小学、初中分批录取 |
| 8月15日前 | 发放入学通知书 |

## 三、小学入学

### 信息登记
- 适龄儿童家长在"上海市义务教育入学报名系统"进行信息登记
- 登记时需携带户口本、房产证等材料
- 人户一致优先安排入学

### 录取排序
各区按照以下顺序统筹安排：
1. 人户一致（户籍地址与居住地址一致）
2. 人户分离（本市户籍，居住地址与户籍地址不一致）
3. 来沪人员随迁子女（持有效期内《上海市居住证》或《居住登记凭证》）

## 四、初中入学

### 信息核对
- 小学毕业生家长在"上海市义务教育入学报名系统"进行信息核对
- 跨区就读学生可回户籍地或居住地所在区就读初中
- 选择回区就读的，需在4月23日前办理相关手续

### 升学方式
- **公办初中**：免试就近入学，按对口地段安排
- **民办初中**：报名人数超过计划数的，实行电脑随机录取

## 五、民办 schools 招生

### 报名条件
- 符合本市义务教育入学条件
- 每位学生可填报2个民办志愿
- 可填报同一学校的2个志愿（如不同校区）

### 录取规则
1. 报名人数 ≤ 招生计划数：全部录取
2. 报名人数 > 招生计划数：电脑随机录取
3. 未被录取的，由区教育行政部门统筹安排公办学校

### 一贯制学校
- 九年一贯制学校可设置直升名额
- 有意愿直升的学生可优先录取

## 六、随迁子女入学

### 申请条件
1. 适龄儿童持有效期内《上海市居住证》或《居住登记凭证》
2. 父母一方持有效期内《上海市居住证》
3. 父母一方在沪缴纳社保满6个月，或灵活就业登记满6个月

### 统筹安排
- 符合申请条件的，由区教育行政部门统筹安排入学
- 优先安排至公办学校就读
- 公办学校学位不足的，安排至政府购买服务的民办学校

## 七、特殊群体保障

### 残疾儿童少年
- 优先安排普通学校随班就读
- 不具备随班就读条件的，安排至特殊教育学校
- 重度残疾的，提供送教上门服务

### 优秀体育学生、艺术骨干学生
- 可跨区报考具有相应招生资格的学校
- 需通过相关资格确认

## 解读与建议

### 家长关注重点

1. **信息登记/核对时间**：4月10日-23日，务必按时完成
2. **人户一致优先**：户籍和房产地址一致的学生优先安排
3. **民办学校摇号**：报名人数超过计划数实行电脑随机录取
4. **一贯制直升**：有意愿直升的学生可优先录取

### 择校建议

- **公办学校**：关注对口地段，提前确认户籍和房产信息
- **民办学校**：合理选择2个志愿，注意未被录取的统筹安排
- **跨区就读**：如需回区，务必在4月23日前办理手续

### 随迁子女

- 提前准备居住证、社保等材料
- 关注各区随迁子女入学具体政策
- 符合条件的可统筹安排至公办或政府购买服务民办学校

### 特殊需求

- 残疾儿童家长可联系区教育行政部门咨询随班就读或特殊教育安排
- 优秀体育学生、艺术骨干学生关注资格确认时间和流程

2026年义务教育招生政策整体稳定，家长需重点关注时间节点和材料准备，确保孩子顺利入学。
