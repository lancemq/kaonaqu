# 2026年上海市普通高校综合评价录取改革试点方案

> **摘要**：上海市教委发布2026年综合评价录取改革试点方案，扩大试点高校范围，优化录取规则。
>
> **来源**：上海市教育委员会  
> **发布日期**：2026-04-01  
> **适用对象**：高中生及家长

---

2026年4月1日，上海市教育委员会发布《2026年上海市普通高校综合评价录取改革试点方案》，对综合评价录取的试点高校、报考条件、校测内容和录取规则进行了详细规定。

## 一、试点高校

### 新增试点高校（2026年）
- 华东理工大学
- 东华大学
- 上海财经大学
- 上海外国语大学

### 原有试点高校
- 复旦大学
- 上海交通大学
- 同济大学
- 华东师范大学
- 上海大学
- 上海中医药大学
- 上海理工大学
- 上海海事大学
- 上海工程技术大学
- 上海师范大学
- 上海中医药大学
- 上海应用技术大学
- 上海电机学院
- 上海商学院
- 上海立信会计金融学院
- 上海第二工业大学
- 上海政法学院
- 上海健康医学院

## 二、报考条件

### 基本条件
1. 符合上海市高考报名条件
2. 综合素质评价信息完整
3. 高中学业水平合格性考试全部合格
4. 符合试点高校招生章程要求

### 优先条件
1. 获得省级及以上学科竞赛奖项
2. 获得省级及以上科技创新奖项
3. 获得省级及以上艺术、体育类奖项
4. 具有突出学科特长
5. 具有突出创新能力
6. 具有突出社会责任感

## 三、校测内容

### 测试形式
- **面试**：主要考查综合素质、创新能力、学科特长
- **笔试**：部分高校设置笔试，考查学科基础
- **体质测试**：所有高校均设置体质测试
- **心理测试**：部分高校设置心理测试

### 测试内容
1. **综合素质**：思想品德、学业水平、身心健康、艺术素养、社会实践
2. **学科特长**：与报考专业相关的学科基础
3. **创新能力**：创新思维、问题解决能力
4. **表达能力**：语言组织、逻辑思维、临场反应

### 体质测试
- **测试项目**：50米跑、立定跳远、坐位体前屈、引体向上/仰卧起坐
- **评分标准**：参照《国家学生体质健康标准》
- **结果使用**：作为录取参考，不计入总分

## 四、录取规则

### 综合成绩计算
- **高考成绩**：85%
- **校测成绩**：15%
- **综合成绩 = 高考成绩（折算为满分660分）× 85% + 校测成绩（满分150分）× 15%**

### 录取流程
1. **高考成绩公布后**：市教育考试院划定综合评价批次最低投档控制分数线
2. **投档**：按综合成绩从高到低投档
3. **录取**：高校根据综合成绩择优录取
4. **公示**：录取名单公示5个工作日

### 志愿填报
- 综合评价批次设置4个平行志愿
- 每个志愿可填报4个专业
- 须选择是否服从专业调剂

## 五、时间安排

| 时间 | 事项 |
|------|------|
| 4月 | 高校发布招生简章 |
| 4月-5月 | 网上报名 |
| 5月下旬 | 校测（面试、笔试） |
| 6月7-9日 | 高考 |
| 6月23日左右 | 成绩公布 |
| 6月下旬 | 综合评价批次志愿填报 |
| 7月上旬 | 综合评价批次录取 |

## 六、重要说明

### 与统一高考的关系
1. **综合评价批次**在本科提前批次之前录取
2. **被综合评价录取的考生**：不再参加后续批次录取
3. **未被录取的考生**：可继续参加后续批次录取

### 校测注意事项
1. **携带材料**：身份证、准考证、综合素质评价报告、获奖证书等
2. **诚信要求**：弄虚作假者取消录取资格
3. **公平公正**：全程录音录像，接受社会监督

### 特殊类型
- **农村学生专项**：部分高校面向农村地区学生设置专项计划
- **学科特长计划**：部分高校针对学科特长突出的考生设置特殊通道

## 解读与建议

### 2026年新变化
1. **试点高校扩大**：新增4所高校，为考生提供更多选择
2. **校测优化**：更加注重综合素质和创新能力考查
3. **体质测试纳入**：所有高校均设置体质测试，考生需重视

### 适合人群
1. **综合素质全面**：不仅成绩好，而且有特长和实践经验
2. **学科特长突出**：在某些学科上有明显优势
3. **创新能力强**：有科技创新成果或实践经历
4. **表达能力强**：面试表现优秀

### 备考建议
1. **提前准备**：关注高校招生简章，了解具体要求
2. **综合素质**：积累社会实践、志愿服务等经历
3. **学科特长**：参加学科竞赛、科技创新活动
4. **面试训练**：模拟面试，提高表达能力
5. **体质锻炼**：保持良好身体素质

### 注意事项
1. **报名材料**：确保真实、完整、及时
2. **校测时间**：合理安排，避免冲突
3. **志愿填报**：合理设置梯度，注意专业调剂
4. **录取后果**：一旦被录取，不得放弃

综合评价录取为综合素质优秀的考生提供了更多机会。建议符合条件的考生积极参加，展示个人优势。
