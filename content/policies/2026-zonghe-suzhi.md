# 上海市初中学生综合素质评价实施办法（2026年修订）

> **摘要**：上海市教委修订初中学生综合素质评价实施办法，明确评价内容、记录方式和在名额分配录取中的使用规则。
>
> **来源**：上海市教育委员会  
> **发布日期**：2026-02-15  
> **适用对象**：初中生及家长

---

2026年2月15日，上海市教育委员会发布《上海市初中学生综合素质评价实施办法（2026年修订）》，对初中学生综合素质评价的内容、记录、使用和管理进行了全面规范。

## 一、评价内容

### 五大维度
1. **品德发展**
   - 理想信念、社会责任、行为习惯
   - 志愿服务、公益活动、集体活动参与情况
   
2. **学业水平**
   - 学习态度、学习方法、学习能力
   - 学业成绩、学科竞赛、创新实践
   
3. **身心健康**
   - 体质健康、心理健康、运动技能
   - 体育锻炼、卫生保健、安全自护
   
4. **艺术素养**
   - 艺术知识、艺术技能、艺术表现
   - 艺术活动、艺术特长、审美能力
   
5. **实践创新**
   - 劳动实践、社会实践、科技创新
   - 研究性学习、发明创造、创业体验

## 二、记录方式

### 写实记录
- **客观记录**：真实记录学生参与活动和取得的成绩
- **证据支撑**：提供相关证明材料（证书、照片、作品等）
- **定期更新**：每学期至少更新一次

### 典型事实
- **突出表现**：记录学生最突出的表现和成果
- **代表性**：选择最能体现学生特点的材料
- **多样性**：涵盖五个维度的代表性事实

### 公示制度
- **校内公示**：每学期公示5个工作日
- **异议处理**：设立异议处理机制
- **最终确认**：公示无异议后录入系统

## 三、系统平台

### 上海市初中学生综合素质评价信息管理系统
- **登录方式**：通过"一网通办"或学校统一账号登录
- **记录主体**：学生自主记录为主，教师指导为辅
- **审核流程**：学生提交→班主任审核→学校公示→录入系统

### 数据使用
- **毕业依据**：综合素质评价合格是初中毕业的必要条件
- **升学参考**：作为高中阶段学校招生录取的重要参考
- **发展指导**：为学生自我认知和未来发展提供依据

## 四、在招生录取中的使用

### 名额分配综合评价录取
- **总分构成**：学业考试总成绩 + 综合素质评价成绩
- **评价成绩**：满分50分，由高中学校根据评价信息评定
- **评定依据**：五个维度的写实记录和典型事实

### 自主招生录取
- **参考因素**：综合素质评价作为重要参考
- **校测内容**：部分学校将综合素质评价纳入校测
- **同等优先**：在同等条件下优先录取综合素质优秀的学生

### 统一招生录取
- **参考因素**：作为投档排序的参考因素之一
- **同分比较**：在投档末位同分时作为排序依据

## 五、评价等级

### 合格标准
- **五个维度**：均达到合格要求
- **记录数量**：每学期至少5条典型事实记录
- **公示要求**：校内公示无异议

### 优秀标准
- **五个维度**：均达到良好以上
- **突出表现**：至少有一个维度表现突出
- **代表性**：典型事实具有代表性

## 六、管理要求

### 真实性
- **严禁造假**：不得弄虚作假、篡改记录
- **责任追究**：发现造假行为，取消相关资格
- **社会监督**：设立举报电话和邮箱

### 公正性
- **程序规范**：严格按照规定程序进行
- **公开透明**：评价标准、过程、结果公开
- **异议处理**：建立完善的异议处理机制

### 安全性
- **数据安全**：保护学生个人信息安全
- **权限管理**：分级授权，责任到人
- **备份机制**：定期备份，防止数据丢失

## 七、时间安排

| 时间 | 事项 |
|------|------|
| 每学期初 | 学生制定学期计划 |
| 学期中 | 记录典型事实 |
| 学期末 | 整理记录、提交审核 |
| 放假前 | 校内公示 |
| 公示后 | 录入系统 |

## 解读与建议

### 2026年修订要点
1. **评价维度优化**：更加注重实践创新和身心健康
2. **记录方式简化**：减少形式化要求，注重实质性内容
3. **使用规则明确**：在名额分配录取中的具体使用方法
4. **监督管理强化**：建立完善的公示和监督机制

### 家长关注重点
1. **及时记录**：提醒孩子及时记录参与的活动和取得的成绩
2. **证据收集**：保留相关证书、照片、作品等证明材料
3. **全面发展**：注重五个维度的均衡发展，不要只关注学业成绩
4. **真实客观**：确保记录真实客观，不弄虚作假

### 学生建议
1. **主动参与**：积极参与各类活动，积累典型事实
2. **及时记录**：养成及时记录的习惯，不要等到期末突击
3. **突出特长**：在某个或某几个维度形成突出表现
4. **定期整理**：每学期整理一次记录，确保完整规范

### 常见误区
1. **只重学业**：忽视其他四个维度的发展
2. **临时突击**：期末集中补录，缺乏过程性记录
3. **虚假材料**：提供虚假证明，影响诚信记录
4. **忽视公示**：不关注公示结果，错过异议处理时间

综合素质评价是学生全面发展的重要体现，也是高中阶段学校招生录取的重要参考。建议家长和学生高度重视，做好日常记录和积累。
