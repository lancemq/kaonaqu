## 政策概览
上海市教委发布2026年特殊教育高中阶段招生入学工作通知，明确招生对象、报名条件、考试安排和录取规则。

## 核心内容
2026年3月10日，上海市教育委员会发布《关于做好2026年上海市特殊教育高中阶段招生入学工作的通知》，对当年特殊教育高中阶段的招生工作进行了全面部署。
## 一、招生对象 ### 基本条件 1. 具有上海市户籍或符合上海市中考报名条件的来沪人员随迁子女 2. 持有残疾人证（视力、听力、言语、肢体、智力、精神、多重残疾） 3. 完成九年义务教育的初中毕业生 4. 年龄在15-18周岁之间 ### 分类招生 - **盲校**：面向视力残疾学生 - **聋校**：面向听力、言语残疾学生 - **培智学校高中部**：面向轻度智力残疾学生 - **普通学校特教班**：面向各类残疾学生 - **随班就读**：符合条件的残疾学生进入普通学校学习 ## 二、报名条件 ### 必备条件 1. 初中毕业或具有同等学力 2. 持有有效的残疾人证 3. 符合上海市中招报名条件 4. 身体健康状况能够适应所报专业的学习要求 ### 特殊要求 - **盲校**：视力残疾，具有基本生活自理能力 - **聋校**：听力或言语残疾，具有基本交流能力 - **培智学校**：轻度智力残疾，具有基本生活自理能力 - **其他**：根据具体学校和专业要求确定 ## 三、考试安排 ### 考试科目 - **文化课考试**：语文、数学、英语 - **专业测试**：根据报考专业确定 - **综合素质评价**：初中阶段综合素质评价结果 ### 考试形式 - **盲校**：盲文试卷或电子试卷 - **聋校**：手语辅助或书面考试 - **培智学校**：简化试卷，延长考试时间 - **其他**：根据残疾类型提供合理便利 ### 考试时间 - **统一考试**：6月20日-21日（与中考同步） - **专业测试**：5月中下旬 - **综合素质评价**：初中阶段记录 ## 四、志愿填报 ### 志愿设置 - **特殊教育学校志愿**：3个平行志愿 - **普通学校特教班志愿**：2个平行志愿 - **随班就读志愿**：1个志愿 ### 填报要求 1. 根据残疾类型选择相应学校 2. 考虑学校专业设置和自身兴趣 3. 了解学校住宿条件和收费标准 4. 填报完成后须进行书面确认 ## 五、录取规则 ### 投档方式 - **平行志愿投档**："分数优先、遵循志愿、一轮投档" - **投档比例**：按招生计划1:1比例投档 - **同分排序**：依次比较语文、数学、英语三科总分 ### 录取批次 1. **特殊教育学校**：优先录取 2. **普通学校特教班**：其次录取 3. **随班就读**：最后安排 ### 特殊照顾 - **残疾等级较重**：适当降低录取分数线 - **生活自理能力强**：优先录取 - **专业特长突出**：破格录取 - **家庭经济困难**：优先安排奖学金和助学金 ## 六、支持政策 ### 教育资助 - **免学费**：所有特殊教育高中阶段学生免收学费 - **助学金**：家庭经济困难学生可申请助学金 - **奖学金**：优秀学生可申请奖学金 - **生活补助**：寄宿生可享受生活补助 ### 学习支持 - **合理便利**：根据残疾类型提供考试和教学便利 - **个别化教育计划**：为学生制定个性化教育方案 - **康复训练**：提供必要的康复训练服务 - **心理辅导**：提供心理健康教育和咨询服务 ### 就业支持 - **职业指导**：提供职业生涯规划和就业指导 - **实习机会**：安排专业实习，提高就业能力 - **就业推荐**：毕业时推荐就业岗位 - **创业支持**：对有创业意愿的学生提供支持 ## 七、时间安排 | 时间 | 事项 | |------|------| | 3月 | 发布招生政策 | | 4月 | 报名申请 | | 5月 | 专业测试 | | 6月 | 统一考试 | | 7月 | 录取工作 | | 8月 | 发放通知书 | ## 解读与建议 ### 2026年新变化 1. **分类招生优化**：更加细化残疾类型与学校匹配 2. **支持政策完善**：教育资助和就业支持力度加大 3. **录取规则人性化**：更加关注学生个体差异 4. **专业设置丰富**：增加适合残疾学生的专业方向 ### 家长关注重点 1. **残疾类型匹配**：选择适合孩子残疾类型的学校 2. **专业选择**：考虑孩子兴趣和未来就业方向 3. **支持政策**：了解可享受的教育资助和便利条件 4. **学校条件**：考察学校教学设施和生活条件 ### 备考建议 1. **文化课复习**：重点复习语文、数学、英语基础知识 2. **专业准备**：根据报考专业进行针对性准备 3. **心理调适**：保持积极心态，树立信心 4. **生活自理**：提高生活自理能力，适应寄宿生活 ### 注意事项 1. **报名时间**：务必在规定时间内完成报名 2. **材料准备**：残疾人证、户口本、初中毕业证等 3. **合理便利**：提前申请考试便利措施 4. **志愿确认**：确认后不得更改 特殊教育高中阶段招生为残疾学生提供了继续深造的机会。
建议符合条件的残疾学生和家长积极关注，做好报名和备考准备。
