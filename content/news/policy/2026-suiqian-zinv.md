## 政策概览
上海市教委发布2026年随迁子女中职自主招生通知，明确适用对象、政策服务要求和招生实施安排。

## 核心内容
2026年3月26日，上海市教育委员会发布《关于做好2026年上海市全日制普通中等职业学校自主招收上海市初中应届毕业生中来沪人员随迁子女工作的通知》，对当年随迁子女中职自主招生工作进行了详细规定。
## 一、适用对象 ### 基本条件 1. 父母一方持有效期内《上海市居住证》 2. 父母一方在沪缴纳社保满6个月，或灵活就业登记满6个月 3. 学生本人在沪参加中考报名 4. 完成九年义务教育的初中毕业生 ### 特殊说明 - 本政策仅适用于报考全日制普通中等职业学校的随迁子女 - 不适用于普通高中和国际课程班招生 - 学生须符合上海市中招报名条件 ## 二、招生学校与专业 ### 招生学校 - **公办中职校**：部分公办中等职业学校 - **民办中职校**：部分民办中等职业学校 - **具体名单**：由市教育考试院统一公布 ### 招生专业 - **中本贯通**：部分专业面向随迁子女开放 - **五年一贯制**：部分专业面向随迁子女开放 - **中高职贯通**：部分专业面向随迁子女开放 - **中职校提前招生**：部分专业面向随迁子女开放 - **普通中专**：大部分专业面向随迁子女开放 ## 三、志愿填报 ### 志愿设置 - **中本贯通**：2个平行志愿 - **五年一贯制和中高职贯通**：5个平行志愿 - **中职校提前招生**：3个平行志愿 - **普通中专**：在统一招生批次填报 ### 填报要求 1. 须符合报考条件 2. 了解学校和专业要求 3. 合理设置志愿梯度 4. 按规定时间完成填报 ## 四、录取规则 ### 投档方式 - **平行志愿投档**："分数优先、遵循志愿、一轮投档" - **投档比例**：按招生计划1:1比例投档 - **同分排序**：依次比较语文、数学、外语三科总分 ### 录取批次 1. **中本贯通** 2. **五年一贯制和中高职贯通** 3. **中职校提前招生** 4. **普通中专** ### 最低投档控制分数线 - **中本贯通**：普通高中最低投档控制分数线 - **五年一贯制和中高职贯通**：相应最低投档控制分数线 - **中职校提前招生**：由学校自主设定 - **普通中专**：由学校自主设定 ## 五、政策服务 ### 咨询服务 - **咨询热线**：12345市民服务热线 - **网上咨询**：上海招考热线网站 - **现场咨询**：各区招考机构 ### 信息服务 - **招生政策**：通过"上海教育"微信公众号发布 - **招生学校**：通过"上海招考热线"网站公布 - **录取结果**：通过"上海招考热线"网站查询 ### 权益保障 - **公平招生**：严禁歧视随迁子女 - **信息公开**：招生政策、计划、结果全面公开 - **申诉渠道**：设立申诉电话和邮箱 ## 六、时间安排 | 时间 | 事项 | |------|------| | 3月下旬 | 发布招生政策 | | 4月 | 随迁子女资格确认 | | 5月 | 志愿填报 | | 6月 | 统一考试 | | 7月 | 录取工作 | | 8月 | 发放通知书 | ## 七、重要说明 ### 与户籍学生的区别 1. **报考范围**：仅限中职校，不能报考普通高中 2. **招生计划**：单列计划，不占户籍学生名额 3. **录取规则**：同等条件下优先录取 4. **收费标准**：与户籍学生相同 ### 注意事项 1. **资格确认**：必须通过随迁子女资格确认 2. **材料准备**：居住证、社保、户口本等 3. **志愿填报**：合理选择学校和专业 4. **录取后果**：被录取后须按时报到 ## 解读与建议 ### 政策意义 1. **教育公平**：保障随迁子女接受中等职业教育的权利 2. **人才培养**：为上海培养技能型人才 3. **社会稳定**：促进来沪人员融入上海 4. **教育服务**：完善随迁子女教育服务体系 ### 家长关注重点 1. **资格条件**：确保符合报考条件 2. **材料准备**：提前准备相关证明材料 3. **学校选择**：了解学校和专业情况 4. **志愿填报**：合理设置志愿梯度 ### 学生建议 1. **认真备考**：努力学习，提高学业成绩 2. **了解专业**：选择适合自身兴趣和能力的专业 3. **积极沟通**：与家长和老师充分沟通 4. **规划未来**：考虑未来发展方向 ### 常见问题 1. **资格确认时间**：通常在4月进行 2. **材料要求**：居住证、社保、户口本、初中毕业证等 3. **志愿填报时间**：与户籍学生同步 4. **录取查询**：通过"上海招考热线"网站查询 随迁子女中职自主招生政策为来沪人员子女提供了更多升学机会。
建议符合条件的学生和家长积极关注，做好报名和备考准备。
