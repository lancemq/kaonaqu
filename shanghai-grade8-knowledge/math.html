一、一次函数

1. 函数概念

  • 在一个变化过程中，有两个变量 x 和 y，如果对于 x 的每一个值，y 都有唯一的值与它对应，那么就说 y 是 x 的函数
  • 自变量：x 的取值范围叫做函数的定义域
  • 因变量：y 的取值范围叫做函数的值域

2. 一次函数解析式

y = kx + b (k ≠ 0)
其中：k 为斜率，b 为截距

3. 一次函数图像与性质

  • k > 0：图像经过一、三象限，y 随 x 增大而增大
  • k < 0：图像经过二、四象限，y 随 x 增大而减小
  • b > 0：图像与 y 轴交于正半轴
  • b < 0：图像与 y 轴交于负半轴
📍 重点：当 k=0 时，y=b 为常数函数（水平直线）

二、勾股定理

1. 勾股定理

a² + b² = c²
（直角三角形两直角边的平方和等于斜边的平方）

2. 勾股定理逆定理

  • 如果三角形三边长 a、b、c 满足 a² + b² = c²，那么这个三角形是直角三角形

3. 常见勾股数

  • 3、4、5
  • 5、12、13
  • 6、8、10
  • 8、15、17

三、平行四边形

1. 平行四边形的性质

  • 对边平行：AB ∥ CD，AD ∥ BC
  • 对边相等：AB = CD，AD = BC
  • 对角相等：∠A = ∠C，∠B = ∠D
  • 对角线互相平分：AO = CO，BO = DO

2. 平行四边形的判定

  • 两组对边分别平行的四边形
  • 两组对边分别相等的四边形
  • 一组对边平行且相等的四边形
  • 对角线互相平分的四边形

3. 特殊平行四边形

  • 矩形：四个角都是直角的平行四边形
  • 菱形：四条边都相等的平行四边形
  • 正方形：四条边相等、四个角都是直角的平行四边形

四、分式

1. 分式的概念

  • 形如 A/B（A、B 是整式，且 B 中含有字母）的式子叫做分式
  • 分式有意义的条件：分母 ≠ 0
  • 分式值为零的条件：分子 = 0 且分母 ≠ 0

2. 分式的基本性质

A/B = (A×C)/(B×C) = (A÷C)/(B÷C) (C ≠ 0)

3. 分式运算

  • 乘法：a/b × c/d = ac/bd
  • 除法：a/b ÷ c/d = ad/bc
  • 加法：a/b + c/b = (a+c)/b

五、数据的分析

1. 集中趋势

  • 平均数：所有数据的和除以数据个数
  • 中位数：将数据从小到大排列，位于中间位置的数
  • 众数：出现次数最多的数据

2. 离散程度

  • 方差：各数据与平均数差的平方的平均数
  • 标准差：方差的算术平方根
