初二化学

沪教版教材重点知识梳理

一、空气和氧气

1. 空气的组成

  • 氮气：78%（N₂）
  • 氧气：21%（O₂）
  • 稀有气体：0.94%
  • 二氧化碳：0.03%
  • 其他气体：0.03%

2. 氧气的性质

  • 物理性质：无色无味的气体，不易溶于水，密度比空气大
  • 化学性质：活泼，能与多种物质反应（氧化性）

3. 氧气的化学性质

  • 与金属反应：3Fe + 2O₂ → Fe₃O₄（火星四射）
  • 与非金属反应：S + O₂ → SO₂（蓝紫色火焰）
  • 与化合物反应：CH₄ + 2O₂ → CO₂ + 2H₂O

4. 氧气的制备

2H₂O₂ → 2H₂O + O₂↑（过氧化氢分解）
2KClO₃ → 2KCl + 3O₂↑（氯酸钾加热）
2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑（高锰酸钾加热）
  • 收集方法：向上排空气法、排水法
  • 检验：带火星的木条复燃

二、水和溶液

1. 水的组成

2H₂O → 2H₂↑ + O₂↑（电解水）
  • 正极产生氧气，负极产生氢气
  • 氢气与氧气体积比为 2:1

2. 水的净化

  • 沉淀：静置，使不溶性杂质沉降
  • 过滤：除去不溶性杂质
  • 吸附：利用活性炭吸附色素和异味
  • 蒸馏：得到纯水

3. 溶液

  • 溶液：一种或几种物质分散到另一种物质里，形成均一的、稳定的混合物
  • 溶质：被溶解的物质
  • 溶剂：能溶解其他物质的物质
  • 饱和溶液：一定温度下，不能继续溶解某种溶质的溶液

4. 溶解度

溶解度 = (溶质质量 / 溶剂质量) × 100g
  • 影响溶解度的因素：温度、溶质和溶剂的性质

三、碳和碳的氧化物

1. 碳的化学性质

  • 稳定性：常温下不易发生化学反应
  • 可燃性：C + O₂ → CO₂
  • 还原性：C + 2CuO → 2Cu + CO₂↑

2. 一氧化碳

  • 物理性质：无色无味的有毒气体，密度略小于空气
  • 化学性质：
    • 可燃性：2CO + O₂ → 2CO₂
    • 还原性：CO + CuO → Cu + CO₂
    • 毒性：与血红蛋白结合

3. 二氧化碳

  • 物理性质：无色无味的气体，密度大于空气，不易溶于水
  • 化学性质：
    • 不支持燃烧
    • 与水反应：CO₂ + H₂O → H₂CO₃
    • 与澄清石灰水反应：CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O

4. 二氧化碳的制备

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
  • 发生装置：固液不加热型
  • 收集方法：向上排空气法
  • 检验：能使澄清石灰水变浑浊

四、酸碱盐

1. 常见的酸

  • 盐酸：HCl，氯化氢的水溶液
  • 硫酸：H₂SO₄
  • 酸的通性：能使紫色石蕊变红，能与金属、金属氧化物、碱、盐反应

2. 常见的碱

  • 氢氧化钠：NaOH，俗称火碱、烧碱
  • 氢氧化钙：Ca(OH)₂，俗称熟石灰、消石灰
  • 碱的通性：能使紫色石蕊变蓝，能与非金属氧化物、酸、盐反应

3. 酸碱中和反应

HCl + NaOH → NaCl + H₂O
  • 酸与碱作用生成盐和水的反应
  • 是复分解反应的一种
